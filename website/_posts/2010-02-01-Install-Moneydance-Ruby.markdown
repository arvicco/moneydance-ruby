## Installing Moneydance Ruby extension

You should have Moneydance 2010 to use this extension.

Download [Ruby Extension](http://github.com/downloads/arvicco/moneydance-ruby/ruby.mxt),
save it as a file "ruby.mxt" on your computer.

Start Moneydance, from "Extensions" menu choose "Add". When "Add Extensions" dialog
opens, tick "From file", click "Next" and choose "ruby.mxt" file that you've downloaded.

Invalid or missing signature warning pops up. This extension is currently under active
development and have not been certified by Moneydance vendor yet.
If you click "Yes", extension will install just fine however.
