moneydance-ruby by Arvicco

MoneyDance extension that adds Ruby scripting support

Moneydance is a full-featured personal financial management application featuring online banking, online bill payment, investment management, budget tracking, scheduled transactions, check printing, detailed reports/graphs and more.

It is cross-platform (Java) and offers best programming API for extension among similar applications. However, currently it only supports Java and Python. This extension adds Ruby support to Moneydance.

Installing extension

You should have Moneydance 2010 to use this extension.

Download Ruby Extension, save it as a file "ruby.mxt" on your computer.

Start Moneydance, from "Extensions" menu choose "Add". When "Add Extensions" dialog opens, tick "From file", click "Next" and choose "ruby.mxt" file that you've downloaded.

Invalid or missing signature warning pops up. This extension is currently under active development and have not been certified by Moneydance vendor yet. If you click "Yes", extension will install just fine however.

Using extension

Once extension is installed, it adds "Ruby Interface" item to Moneydance Extension menu. Clicking this item starts Ruby (IRB) console that you can use to run Ruby code against live Moneydance framework and data collections. You can load/execute Ruby source files from console too, using "Load file" option.

You can also run your Ruby scripts inside Moneydance environment via command line like this (Unix): 

$ moneydance moneydance:fmodule:ruby:file?/path/to/script.rb

For OSX, your line will be like this (assuming you installed Moneydance into Applications): 

$ /Applications/Moneydance.app/Contents/MacOS/JavaApplicationStub \
      -invoke_and_quit moneydance:fmodule:ruby:file?/path/to/script.rb

You can read a Primer on JRuby and Moneydance Methods that highlights some of the useful methods and approaches to programmatically accessing Moneydance. Possibilities are limitless, this is Ruby - play with data models interactively, read the reference docs and hack away.

Building from source

This is for those who enjoy doing things their own way. You should have Java 1.6 and JRuby 1.5+ installed and functional to build extension from source.

Download this project in either ZIP or TAR formats, unzip/untar it (say, to "moneydance-ruby" dir).

You can also clone the project with Git by running: 

$ git clone git://github.com/arvicco/moneydance-ruby

Change to unzipped source code dir, run Rake task that builds the extension. 

$ cd moneydance-ruby 
$ rake ant:jar

This should compile and build "ruby.mxt" extension file inside "dist" directory. You can then install it into Moneydance using technique described in "Installing extension" section above.
